# Psych Directory Instructional Guidelines

**This Directory is not for the weak I have spent over two weeks researching the best methods to implement realistic memory development algorithms for the models development**

## What is in the Directory

> Below you will find all the algorithms which utilized to develop multiple human-like learning mechanics within the machine
The algorithms are as follows:
    Bayesian knowledge tracing which is a probabilistic framework which will model students Mastery as a hidden Markov process we will track binary learning States using four separate core parameters:
P(L<sub>0</sub>) Represents initial probability of knowledge
P(T) represents probability of learning after an opportunity
P(S) represents probability of a slip or careless error
P(G) represents probability of a correct guess
On a secondary note for long-term memory retention we will use Advanced spaced repetition algorithms utilizing the free spaced repetition scheduler. in order to do this the profiles we have created will be tracking retrievability metrics so when the prompt converter is called It Can instruct the llm API to seamlessly interleave past topics into the current session, only when the subject is matching. in the future however I would like to have professors or teachers determine when this occurs to prevent random event calls.

 zone of proximal development and cognitive load theory in order to record Dynamic scaffolding levels to ensure the LM provides hints that strike the exact balance between productive struggle and cognitive overload
